important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface for web documents.

How do you select an element from the DOM?

For a single element I use getElementById() or querySelectorAll() method.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a parent element listens for events on its child elements, leveraging event bubbling. It is useful because it reduces memory usage and also reduces redundancy and simplifies code.

How do you manipulate an element's attributes and styles using the DOM?

By using getAttribitue() and setAttribute() method.
